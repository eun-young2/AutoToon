# 📎 Auto Toon(팀명: 3큼폭발)

![서비스이미지-003](https://github.com/user-attachments/assets/e5aaeb48-2b41-47b1-9161-83266d1b9b36)





## 👀 서비스 소개
* 서비스명: Auto Toon
* 서비스설명: 회원이 일기를 작성하면 일기에 맞는 네컷 만화를 생성해준다.
<br>

## 📅 프로젝트 기간
2025.05.01 ~ 2025.06.17 
<br>

## ⭐ 주요 기능
* 일기작성하기
* 일기 작성하기
* 일기 수정하기
* 네컷만화 생성하기
* 캘린더에서 일기 상세보기
<br>
